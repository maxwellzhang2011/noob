# noob
a neovim package

# installing
to download the packat use this command
`git clone https://github.com/maxwellzhang2011/noob.git`
then to install neo-tree do
`./install.sh ntre_install`
if you already have neo-tree or just updating just use
`./install.sh noob_install`
if your just starting neovim use
`./install.sh full`
double check if this is the version you want
`./install.sh version`
